Why use Linux

Isn't it time to switch to Linux? 18 reasons to leave Windows

BOTTOM  |  TOP

  1. Linux is free.

  2. Linux is fast.

  3. Linux is gratis.

  4. Linux is stable.

  5. No virusses.

  6. No serial numbers.

  7. No malicious software (malware).

  8. There are companies that do not allow the use of Windows for the above reasons, see The Register.

  9. Misconceptions About Open Source Software (Don't Judge a Program By Its Software License), see How-To Geek.

  10. Good language support.

  11. Boot from a Live USB stick (or DVD) and check that everything works before installing.

  12. Easy installation in just a few guided steps.

  13. Linux also runs on older hardware, no need to (always) buy a new computer.

  14. Performing updates is system-wide; the entire operating system and all software.

  15. Install software with one click, from over 69,000 packages, something for everyone.

  16. Lots, lots of help on the Internet, for example via Debian User Support and Ask Ubuntu.

  17. Various desktop environments available such as GNOME, Xfce, and KDE.

  18. You can work as usual with Office (LibreOffice), Instant Messaging (Pidgin), Web (Google Chrome or Mozilla Firefox), and Mail (Thunderbird), see also Open Source for Windows and Open Source for Mac.
