Script kz-get downloads and installs Debian or RPM package kz.
For how to use this script see 'Checklist installation' above.
Debian package kz contains scripts for installing and managing Debian and Debian-based systems such as Ubuntu.
Install with: sudo apt install wget; wget karelzimmer.nl/getkz; bash getkz
For how to use this package see 'Checklist installation' above.
RPM package kz contains scripts for installing and managing Red Hat and Red Hat-based systems such as Rocky Linux.
Install with: sudo dnf install wget; wget karelzimmer.nl/getkz; bash getkz
For how to use this package see 'Checklist installation' above.
