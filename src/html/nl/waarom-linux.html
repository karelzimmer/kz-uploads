Waarom Linux gebruiken

Is het geen tijd om over te stappen naar Linux? 18 redenen om Windows te verlaten

ONDERKANT  |  BOVENKANT

  1. Linux is vrij.

  2. Linux is snel.

  3. Linux is gratis.

  4. Linux is stabiel.

  5. Geen virussen.

  6. Geen serienummers.

  7. Geen kwaadaardige software (malware).

  8. Er zijn bedrijven die het gebruik van Windows om bovenstaande redenen niet toestaan, zie The Register (Engels).

  9. Misvattingen over open source-software (Beoordeel een programma niet op basis van zijn softwarelicentie), zie How-To Geek (Engels).

  10. Goede taalondersteuning, 99.9% in het Nederlands.

  11. Start op vanaf de Ubuntu of Debian Live usb-stick (of dvd) en controleer eerst of alles werkt alvorens te installeren.

  12. Makkelijk installeren in slechts enkele stappen, met name bij Ubuntu en Debian.

  13. Linux draait ook op oudere hardware, niet nodig om (steeds) een nieuwe computer te kopen.

  14. Updates uitvoeren geldt voor het hele systeem; het hele besturingssysteem en alle software.

  15. Installeer software met één muisklik, een keuze uit meer dan 58.000 pakketten, voor elk wat wils.

  16. Veel, heel veel, hulp op het Internet, bijvoorbeeld via Ubuntu Nederland en Ubuntu Documentatie in het Nederlands, en Ask Ubuntu in het Engels.

  17. Diverse bureaubladomgevingen beschikbaar zoals GNOME in Ubuntu, KDE in Kubuntu, Xfce in Xubuntu, MATE in Ubuntu MATE, of LXDE in Lubuntu.

  18. U kunt werken zoals u gewend bent met Office (LibreOffice), IM (instant Messaging) (Pidgin), Web (Firefox), en Mail (Thunderbird), zie ook Open Source voor Windows en Open Source voor Mac.
