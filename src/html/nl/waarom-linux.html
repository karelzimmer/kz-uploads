Waarom Linux gebruiken

Is het geen tijd om over te stappen naar Linux? 20 redenen om Windows te verlaten

ONDERKANT  |  BOVENKANT

  1. Linux is vrij (betere controle door de gebruiker).

  2. Linux is snel.

  3. Linux is gratis.

  4. Linux is stabiel.

  5. Geen serienummers.

  6. Geen licentiekosten.

  7. Geen virussen.

  8. Geen kwaadaardige software (malware).

  9. Geen advertenties en spyware (verhoogde privacy).

  10. Goede taalondersteuning, 99.9% in het Nederlands.

  11. Eenvoudige installatie in slechts een paar begeleide stappen.

  12. Start op vanaf een live usb-stick (of dvd) en controleer eerst of alles werkt alvorens te installeren.

  13. Linux draait ook op oudere hardware, niet nodig om (steeds) een nieuwe computer te kopen (goed voor de planeet).

  14. Updates uitvoeren geldt voor het hele systeem; het hele besturingssysteem en alle software.

  15. Installeer software met één muisklik, een keuze uit meer dan 69.000 pakketten, voor elk wat wils.

  16. U kunt werken zoals u gewend bent met Office (LibreOffice), Instant Messaging (Pidgin), Web (Google Chrome of Mozilla Firefox), en Mail (Thunderbird), zie ook Open Source voor Windows en Open Source voor Mac.

  17. Diverse bureaubladomgevingen beschikbaar zoals GNOME (meest gebruikt), Xfce (voor oude computers), en KDE (lijkt het meest op Windows).

  18. Veel, heel veel hulp op het Internet, bijvoorbeeld via Debian Gebruikersondersteuning en Ask Ubuntu in het Engels.

  19. Er zijn bedrijven die het gebruik van Windows niet toestaan, zie The Register (Engels).

  20. Misvattingen over open source-software (Beoordeel een programma niet op basis van zijn softwarelicentie), zie How-To Geek (Engels).
